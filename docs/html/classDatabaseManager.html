Descripción detallada

Clase que administra la conexión y las operaciones CRUD sobre la base de datos de componentes.

Esta clase hereda de QObject para integrarse con el sistema de señales y ranuras de Qt. Permite abrir/cerrar la base de datos, inicializar tablas y realizar operaciones de inserción, actualización, eliminación y consulta de componentes.
